Detailed Description

template<bool Const, typename ValueT>
class GenericArray< Const, ValueT >

Helper class for accessing Value of array type.

Instance of this helper class is obtained by GenericValue::GetArray(). In addition to all APIs for array type, it provides range-based for loop if CEREAL_RAPIDJSON_HAS_CXX11_RANGE_FOR=1.
